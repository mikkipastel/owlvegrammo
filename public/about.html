About

โปรเจกนี้ ใช้สำหรับร่วมแข่งขัน เอ้ยยย ร่วมสนุกในงาน PWA online hackathon ซึ่งเป็นงาน hackathon ที่สองของคนทำ แถมทำคนเดียวด้วย (ชวนเพื่อนแล้วไม่มีใครมาสักคน) ตอนแรกยังหาไอเดียไม่ได้เลยว่าจะทำอะไรดี เราจะทำไหวไหม เพราะเราเป็น Android Developer ไง ไม่ได้มาทางสายเว็บ พอเขียน HTML กับ CSS ได้บ้าง แต่ js ก็ทำจริงจังงานนี้งานแรกแหละ


owlvegrammo เดิมทีมาจากแพลนที่เราจะหาเงิน เอ้ยยย เปิดสอนโปรแกรมมิ่งแบบเล็กๆ ชื่อว่า owlve programming school เพื่อพัฒนาน้องๆหนูๆเข้าสู่วงการต่อไป ทีนี้มีข้อจำกัดด้านการสอน เพราะเราเองไม่มีเวลาเตรียมอะไรขนาดนั้น งานประจำก็มี บล็อกตัวเองก็เขียนไม่ทันอีก แต่เราเขียนสรุปลำดับเนื้อหาไปแล้วแต่คิดว่าถ้าเขียนบล็อกแต่ละหัวข้อ แล้วอ่านหัวข้อต่อไปมันจะเสียเวลาไปไหม มีข้อจำกัดถ้า internet ไม่มี ดังนั้นจึงทำสิ่งนี้ใน PWA online hackathon เพราะ solution นี้น่าจะตอบโจทย์ ไม่ต้องทำทุก platform ก็สามารถใช้งานได้ง่ายเหมือนลงแอป กิน resource ตามจริง สามารถ offline ได้บางส่วน มีไอคอนสวยงาม


owlvegrammo ได้ไอเดียจากหลายๆเว็บ หลายๆแอป ในเรื่องหน้าตา ส่วนไอเดียหลักๆในเรื่องวิธีใช้ก็มาจาก Primer แอปสอน online marketing ของ google ที่สามารถตอบโจทย์การเรียนรู้ในเรื่องต่างๆ ในเรื่องของการเรียงลำดับเนื้อหาของแต่ละบท การข้ามไปอ่านหน้าถัดไป ดังนั้น owlvegrammo จึงเป็นการสรุปเนื้อหามาเป็นลำดับขั้นตอน เพื่อให้อ่านทำความเข้าใจได้ง่ายขึ้น เสมือนเป็น short-note ก็ว่าได้


เนื้อหาใน owlvegrammo มีดังนี้ ใน first version และเนื้อหา แน่นอน เป็นภาษาไทยทั้งหมด!
